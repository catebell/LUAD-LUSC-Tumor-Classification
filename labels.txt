PROCEDIMENTO PER CREAZIONE GRAFO:
- (mRNA <--> mRNA) correlazione biologica (co-expression) tra i geni, considerando tutti i pazienti o no, db STRING e WGCNA
- (CNV --> mRNA) in base al gene_id
- (methylation --> mRNA) in base al gene_id  [se b-value alto e mRNA alta espressione --> gene represso]

A graph for each patient, then graph classification.


RNA:
  WGCNA
   - Open files from directory RNA (ignore comment # and metadata),
        keep just gene_type = protein_coding and drop duplicates based on gene_name,
        remove .version from gene_id
   - Use file_case_mapping.tsv to match the case_id with the filename (take omic = RNA)
   - Build expression matrix with genes as rows (gene_id) and samples as columns (case_id)
   - Fill the matrix with the values log2(TPM + 1) (tpm_unstranded)
   - Filter the genes less expressed
   - Substitute the column gene_id with gene_name (otherwise STRING couldn't recognize them)
   - Create a CSV for STRING (https://string-db.org/cgi/input?sessionId=bwrTOSYjtgG7&input_page_active_form=multiple_identifiers)
  STRING
   - Upload the CSV created and select Organisms: Homo sapiens, Confidence score: high 0.7
   - Creates a graph PPI (protein-protein interactions)

METHYLATION:
   - Open files from directory methylation (.txt so header = None),
        transform them into .csv with 2 columns: cpg_IlmnID and beta_value
   - Use file_case_mapping.tsv to match the case_id with the filename (take omic = methylation)
   - Use matched_cpg_genes_converted.csv to match cpg_IlmnID
   - Set the promoter_upstream = 1500 and the promoter_downstream = 500
   - Select CpG in the promoter window, using:
        - start = [gene] - promoter_upstream
        - end = [gene] + promoter_downstream
        substitute [gene] with: if gene_strand = + use gene_start while if gene_strand = - use gene_end
        if start <= cpg_island <= end it's in the promoter so take only these
   - Compute average beta value per gene [0 - 1]
        {
          - average beta value >= 0.6 methylated -> gene expression repressed
          - average beta value <= 0.3 unmethylated -> gene expression active
        }
   - Save this value to add to the nodes of the graph
   - Use Pearson to compute correlation between beta and gene expression
        if the edge already exists increase its weight otherwise create it
        {
          repression isn't in the node but in the relation beta->expression so use Pearson
          gene repressed if high average beta value and correlation(beta, expression) < 0
        }

CNV:
   - Open files from directory CNV and remove .version from gene_id
   - Use file_case_mapping.tsv to match the case_id with the filename (take omic = CNV)
   - Convert to numeric the columns copy_number, min_copy_number and max_copy_number (NaN for missing values)
   - Compute the diff = max - min, only if min and max exist
        {
          - copy_number = 2: normal state
                - if diff = 0: neutral -> normal gene
                - if diff != 0: complex rearrangement (common in tumors) -> possible oncogene activation
          - copy_number < 2 (so, 0 and 1): deletion
                - if diff = 0: whole-gene deletion -> TP53, CDKN2A absent (complete deletion of these genes causes tumor progression)
                - if diff != 0: partial deletion / intragenic -> gene fragments, loss of function
          - copy_number > 2 (so, 3, 4, ...): duplication
                - if diff = 0: whole-gene gain -> EGFR (LUAD), MET amplified (increase gene expression and tumor growth, used as biomarkers)
                - if diff != 0: partial gain / focal amplification -> selective increase of functional domains
        }
   - Save these values to add to the nodes in the graph
   - Use Pearson to compute correlation (CNV -> RNA)

Obtain Multi-omics graphs and save them as Dataset objects into data_graph_processed/:
    [multiomics_graph_creation.py, PatientGraphDataset.py]
    h0 -> [num_nodes, 3]
        - Nodes = genes, features = [RNA, methylation, CNV]
        - Edges = biologic network (PPI, ...), features = [avg_combined_score, max_combined_score, num_protein_links]

GIATConv:
    h1 = GIATConv(h0) -> LP: h1 = LabelPropagation(h1) -> [num_nodes, hidden * heads]
    h2 = GIATConv(h1) -> LP: h2 = LabelPropagation(h2) -> [num_nodes, hidden * heads]

G concatenation (Jump Knowledge):
    G = concat(h0, h1, h2) -> [num_nodes, 3 + hidden * heads + hidden * heads]
    graph embedding = global_pool(G) -> [batch_size, graph_dim]

MLP encoder:
    v_graph = post_pool_MLP(graph_embedding) -> [batch_size, 128]
